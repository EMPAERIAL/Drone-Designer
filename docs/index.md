# Drone Designer Documentation

Drone Designer is a Windows desktop application that turns mission requirements into a candidate multirotor design. It selects components from the project catalogue, presents the resulting configuration in the UI, and can drive downstream export and CAD-generation workflows.

This documentation set is split by audience. Use the section below that matches what you are trying to do.

## Start Here

### I want to use Drone Designer

Go to [User Documentation](user/index.md) if you need to:

- install or launch the packaged application
- design a UAV from mission requirements
- export results or generate outputs
- troubleshoot common operator-facing problems

### I want to maintain or extend Drone Designer

Go to [Maintainer Documentation](maintainers/index.md) if you need to:

- understand how the documentation set is organized
- onboard an AI agent or human maintainer
- navigate architecture, concepts, workflows, and repo terminology
- find sanctioned change paths for common maintenance tasks

## Documentation Map

- [User Documentation](user/index.md): operator-facing installation, workflows, and troubleshooting.
- [Maintainer Documentation](maintainers/index.md): onboarding, glossary, architecture, concepts, and maintenance workflows.
- [Reference Documentation](reference/index.md): stable facts about the component database, runtime dependencies, and validation.
